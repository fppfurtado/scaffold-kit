# ADRs (Architecture Decision Records)

Registro imutável de decisões técnicas importantes.

Use apenas quando uma decisão realmente impacta o projeto a longo prazo (ex: escolha de LLM, banco de dados, arquitetura).

**Formato padrão:** `ADR-0001-nome-da-decisao.md`

Cada ADR deve ser curto e conter:
- Contexto
- Decisão
- Alternativas consideradas
- Prós e contras
- Data
