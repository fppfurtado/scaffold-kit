# Pitches

Aqui ficam as **propostas de ciclos** de desenvolvimento (normalmente 4–6 semanas).

Cada arquivo representa uma visão clara do que será entregue em um ciclo.

**Como criar um pitch:**
- Copie o template de `cycles/cycle-01/pitch.md`
- Defina objetivo, escopo, riscos e estimativa
- Após aprovação, inicie o ciclo correspondente
